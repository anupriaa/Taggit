Getting Started



How to save to Taggit?

Saving to Taggit is super simple. Just Add the bookmarklet to your browser and click it each time you want to save a web page. Once saved you can access it through the application.



Where can the bookmarklet be found?

The bookmarklet is available under 'Add Bookmarklet' tab. Once logged in, this tab is available.



What can be saved on Taggit?

As of now just urls can be saved, either manually or through the bookmarklet.



How can the saved data be accessed?

The saved data is available on the web application. Once the user signs in, the application takes the user straight to search page.The user can search using keywords.



The user has to remember the keywords?

No. The user does not even enter the keyword. Taggit does all of it for the user. The user can search with words associated with the web page and taggit will find it.
